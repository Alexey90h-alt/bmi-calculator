# ⚖️ BMI Calculator (Калькулятор ИМТ)

Простое приложение на **Python** для расчета индекса массы тела (ИМТ). Проект создан в учебных целях для отработки навыков работы с математическими формулами и условиями. 

Насколько этим цифрам можно верить?
Как алгоритм — расчет точен. Но как медицинский показатель, ИМТ имеет свои нюансы, о которых стоит знать:

Мышцы тяжелее жира: Если ты профессиональный атлет с горой мышц, программа может сказать «Избыточный вес», хотя на самом деле у тебя просто много мышечной массы.

Тип телосложения: ИМТ не учитывает ширину кости или распределение жира (живот vs бедра).

Возраст: Для детей и пожилых людей нормы ИМТ немного отличаются от стандартных «взрослых».

Итог: Для обычного человека это отличный быстрый ориентир. Но если ты Шварценеггер, то программа впадет в ступор!

## 🧮 Как это работает
Программа использует стандартную формулу Кетле:
$$BMI = \frac{weight}{height^2}$$

### Таблица интерпретации результатов:
| ИМТ | Категория |
| :--- | :--- |
| < 18.5 | Недостаточный вес 🍎 |
| 18.5 – 24.9 | Норма ✅ |
| 25.0 – 29.9 | Избыточный вес 🥗 |
| > 30.0 | Ожирение 💪 |

## 🛠️ Что внутри
* **Ввод данных:** Работа с типом `float` для точности.
* **Математика:** Возведение в степень и округление `round()`.
* **Логика:** Разветвленные условия `if-elif-else`.
* **Стиль:** Переменные на английском языке.
